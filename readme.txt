== Description ==


Automatically prompt subsription message to promote and make visitor subscribe your feed every they post a new comment

It uses both FeedBurner and FeedBurner's Mail Subscriptions service. For more info visit <a href="http://mr.hokya.com/auto-subscribe-rss-feed/">here</a>


== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
1. Activate the plugin through the 'Plugins' menu in WordPress
